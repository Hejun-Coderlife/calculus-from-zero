从 0 开始的微积分 · 第四章
怎样围出最大的地？
一段固定长的篱笆（比如 20 米），围成一个矩形。
形状随你变——瘦高、矮胖……
哪一种，圈住的面积最大？
有个最高点
面积，先涨后跌
把宽从小拖到大：面积先变大，到某处最大，再变小。
中间藏着一个顶点。
怎么精确找到它？—— 用上一章学的坡度（导数）。
找顶点
坡度 = 0 的地方
画出「面积随宽变化」的曲线。顶点左边面积还在涨（坡度 > 0）；右边开始跌（坡度 < 0）；正顶点那一刻，坡度 = 0（切线水平）。移动宽，看坡度什么时候变成 0。
用起来
山顶、谷底，都在「切线水平」的地方 —— 也就是坡度 = 0、导数 = 0。
所以，要找一个东西的最大或最小，就去找导数 = 0 的地方：
导数 = 0切线水平的那一点→最大 / 最小山顶 或 谷底
解出来
答案是正方形
回到篱笆题：宽 + 高 = 10（周长一半），面积 = 宽 × 高。
在上一屏把宽拖到坡度正好 = 0，读到的是 宽 = 5。
这时高也 = 5 —— 正方形，面积最大 = 25。
✅ 周长一样时，越接近正方形，面积越大。谷底（最小）同理：也是坡度 = 0 的地方。
导数的超能力
不只是「变多快」，
还能找最好。
第二章：导数告诉你「某一刻变多快」。
这一章：导数 = 0，帮你锁定「山顶谷底」—— 最大利润、最省材料、最短时间，全靠它。
这就是优化。
