从 0 开始的微积分 · 第五章
别拖了，能算吗？
前面几章，我们靠拖滑块，一点一点「看」出曲线在某处有多陡 —— 那个「陡」，就是导数。
可每换一个点，就得重拖一次，太笨了。
能不能不拖也能算，直接得出任意一点的导数？
放大的魔法
弯的，放大到极致会变直
把曲线在某一点无限放大，弯会越来越不明显，最后看起来就是一条直线。
那条直线的斜率，就是这一点的导数。
—— 于是「算导数」变成了「量那条直线的斜率」。
放大，量斜率
在 x = a 处，放大
这是 y = x²。把窗口越缩越小（放大倍数越拉越大），曲线肉眼就变直了。
在小三角形上量斜率 = Δ竖 ÷ Δ横；放大越狠，这个数越稳。
再拖动 a，看这个「稳下来的斜率」怎么随 a 变。
找规律
换几个函数，都放大量一量
同样的招：放大、量斜率。换 x²、x³、√x 试试，把每个的导数记下来 —— 一张表就露出规律。
斜率（也叫导数）：x² → 2x　·　x³ → 3x²　·　√x = x½ → ½·x−½
为什么 √x 的斜率 = ½·x−½ ？—— 点开看推导（用 x=4）
给它一个规则
看出来了吗？指数搬下来当系数，指数再减 1。这就是幂法则：
xn一个幂函数→n·xn−1指数 n 搬到前面当系数
指数减 1
验一下：x²（n=2）→ 2·x¹ = 2x ✓　　x³（n=3）→ 3·x² ✓
为什么对所有 n 都成立？—— 点开看证明
从此不用拖了
导数，一步算出
还记得第一屏那道题吗？—— y = x² 在某点，斜率是多少？那时只能拖滑块、量半天。
现在有了幂法则：x² 的斜率 = 2x。想知道哪点，把那点的 x 代进去，一步就出 👇
拖滑块换任意 x —— 斜率永远 = 2×那个 x，一步算出。
第三、四屏是「拖两点、逼近着量」；这里是「代公式、直接算」—— 这就是会求导的意义。
🧰 你多了一件工具：幂法则 —— 不用再拖滑块，看一眼就写出导数。这也是「求导公式」的第一块基石。
